Vision-Based Gesture Controlled Robotic Arm

A robotic arm controlled using real-time hand gesture recognition with OpenCV, MediaPipe, Python, and Arduino Nano.

Project Overview

This project uses computer vision and hand tracking to control a robotic arm through natural hand gestures without any wearable devices. The system detects hand landmarks using MediaPipe and translates gestures into robotic arm movements through Arduino Nano and servo motors.

Technologies Used

Python

Project Features

  • Real-time hand gesture recognition
  • Robotic arm movement control
  • Serial communication using PySerial
  • Servo motor actuation
  • Contactless human-machine interaction
  • Vision-based gesture tracking
  • Arduino Nano motor control
  • Multi-servo robotic movement
